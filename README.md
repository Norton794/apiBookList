<h3 id="como-usar">Como executar o projeto</h3>

```bash

# Clone este repositório
$ git clone <https://github.com/Norton794/apiBookList/>

# Instale as dependências
$ npm install

# Execute a aplicação 
$ nodemon index.js

# O servidor inciará na porta:8083 - acesse <http://localhost:8083> 

```
